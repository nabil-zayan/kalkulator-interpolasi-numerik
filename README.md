# Kalkulator Interpolasi Polinomial

Aplikasi web interaktif untuk melakukan kalkulasi interpolasi polinomial (**Linear, Kuadratik, dan Kubik**) secara presisi. Proyek ini disusun untuk memenuhi tugas mata kuliah **Metode Numerik**.

## 🚀 Fitur Utama
- **Algoritma Numerik Stabil:** Menggunakan pembentukan **Matriks Vandermonde** yang diselesaikan melalui **Eliminasi Gauss** dengan *partial pivoting* untuk akurasi tinggi.
- **Visualisasi Dinamis:** Integrasi **Chart.js** untuk menampilkan kurva polinomial, titik data asli, dan titik hasil prediksi secara *real-time*.
- **Antarmuka Modern:** Desain antarmuka yang bersih, responsif, dan *user-friendly* dengan tipografi monospaced untuk kemudahan pembacaan angka.
- **Sistem Validasi:** Dilengkapi dengan pengecekan data untuk mencegah input kosong atau duplikasi nilai X.
